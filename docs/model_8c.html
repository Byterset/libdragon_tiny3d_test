void model_load ( struct model *  into,
const char *  model_filename 
)

Load a T3D model from a file and store it in the model struct. Also load the skeleton if the model has one. Depending on if a Skeleton exists, the rspq_block will be generated to draw the model skinned or unskinned. The generated RSPQ Block is a default t3d_model_draw block, to customize the drawing, create a callback to the custom draw function and add it to the renderbatch.
