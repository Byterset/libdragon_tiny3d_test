This function calculates a value that simulates a bounce-\/back effect based on the input parameter t. The result is derived from the equation -\/t + t$^\wedge$2, which creates a parabolic curve that starts at 0, rises to a peak, and then falls back to 0.

Moves a value towards a target value by a maximum amount. 

This function calculates a new value that is moved from the starting value ({\ttfamily from}) towards the target value ({\ttfamily to}) by a maximum amount ({\ttfamily max\+Move}). If the difference between {\ttfamily from} and {\ttfamily to} is less than or equal to {\ttfamily max\+Move}, the function returns the target value ({\ttfamily to}).
