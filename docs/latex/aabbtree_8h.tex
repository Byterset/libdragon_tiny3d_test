Moves a node within an \mbox{\hyperlink{structAABB}{AABB}} tree and updates its \mbox{\hyperlink{structAABB}{AABB}}. 

The displacement vector is used to enlarge the node bounds by anticipated future movement and reduce the number of leaf insertions. To avoid unnecessary tree updates, the node is only moved if the new \mbox{\hyperlink{structAABB}{AABB}} does not fit within the old \mbox{\hyperlink{structAABB}{AABB}}. To disable the displacement feature, set the displacement vector to zero.
