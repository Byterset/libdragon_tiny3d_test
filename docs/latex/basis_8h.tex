\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{structBasis}{Basis}} structure and functions for quaternion-\/based rotations. 

This header file contains the definition of the \mbox{\hyperlink{structBasis}{Basis}} structure, which represents a 3D coordinate system using three vectors (x, y, z). It also provides functions to initialize a \mbox{\hyperlink{structBasis}{Basis}} from a quaternion and to perform rotations and inverse rotations using the \mbox{\hyperlink{structBasis}{Basis}}.
