Initializes a raycast with the given parameters and returns a new raycast struct.

Parameters
originthe origin point of the ray in world space
dirthe direction of the ray (will be normalized during initialization)
maxDistancethe maximum distance until which hits will be considered. Must be > 0
maskdecides if the ray will hit physicsObjects, static Collision or both
interact_triggertoggle if the raycast will be able to hit physicsObjects that are triggers
collision_layersthe mask of the rays collision layers - the layers the ray will be able to hit
ignore_layersthe mask of the collision layers that will be explicitly ignored by the raycast
Returns
a new initialized raycast structure
