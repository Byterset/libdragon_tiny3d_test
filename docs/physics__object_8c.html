Parameters
entity_idThe entity ID of the owner entity of this physics object.
objectPointer to the object to be initialized.
collisionPointer to the objects collision information. Must be set!
collision_layersFlags that define which layers the object can collide/interact with. See enum collision_layer
positionPointer to the 3D vector that represents the objects position (usually shared with the owners). Must be set!
rotationPointer to the quaternion rotation that represents the objects rotation (usually shared with owner). May be NULL.
center_offset3D vector that describes how the phys objects collision is offset from the position (eg if the model orign is not at center)
massThe mass of the physics object. Must be > 0.
