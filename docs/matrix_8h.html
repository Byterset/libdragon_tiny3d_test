Computes the normalized Z value for a given depth within a specified near and far plane.

This function calculates the normalized Z value based on the depth, near, and far plane distances. The normalized Z value is used to map depth values to a normalized range.

Parameters
depthThe depth value to be normalized.
nearThe near plane distance.
farThe far plane distance.
Returns
The normalized Z value. Returns -1.0 if the depth is greater than or equal to -near, 1.0 if the depth is less than or equal to -far, otherwise returns the computed normalized Z value.
