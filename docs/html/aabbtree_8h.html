int AABBTree_moveNode ( AABBTree *  tree,
NodeProxy  node,
struct AABB  aabb,
struct Vector3 *  displacement 
)

Moves a node within an AABB tree and updates its AABB.

The displacement vector is used to enlarge the node bounds by anticipated future movement and reduce the number of leaf insertions. To avoid unnecessary tree updates, the node is only moved if the new AABB does not fit within the old AABB. To disable the displacement feature, set the displacement vector to zero.

Parameters
treeThe AABB tree.
nodeThe index of the node to move.
aabbThe new AABB for the node.
displacementThe displacement vector to move the node. Difference between the old and new position
Returns
Returns TRUE if the node was moved, FALSE otherwise.
