void quatDecompose ( struct Quaternion *  input,
struct Vector3 *  axis,
float *  angle 
)

Decomposes a quaternion into an axis and an angle.

Parameters
inputPointer to the input quaternion to be decomposed.
axisPointer to the vector that will store the axis of rotation.
anglePointer to the float that will store the angle of rotation in radians.

The function calculates the magnitude of the quaternion's vector part and normalizes it to obtain the axis of rotation. If the magnitude is very small (less than 0.0001), it sets the axis to a default up vector and the angle to 0. Otherwise, it normalizes the axis and calculates the angle using the sine of the magnitude.
