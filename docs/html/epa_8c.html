Solves the Expanding Polytope Algorithm (EPA) to find the penetration depth and contact normal between two objects.

This function takes an initial simplex from the GJK algorithm and iteratively expands it to find the closest face to the origin. It then calculates the penetration depth and contact normal between the two objects.

Parameters
startingSimplexA pointer to the initial simplex obtained from the GJK algorithm.
objectAA pointer to the first object involved in the collision.
objectASupportA function pointer to the GJK Support function for the first object.
objectBA pointer to the second object involved in the collision.
objectBSupportA function pointer to the GJK Support function for the second object.
resultA pointer to an EpaResult structure where the result will be stored.
Returns
true if the EPA algorithm successfully finds a penetration depth and contact normal, false otherwise.
